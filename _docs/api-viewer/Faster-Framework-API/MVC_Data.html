		getNewID (line 1178)
	
 
	
	

A handy class method to give you a unique ID if you don't like a database default autonumbering.


Some people don't like database default autonumbering because programmers can mess up the data  integrity if they don't know what they are doing in transferring a database. By using these keys  instead, the data integrity remains intact.

	
    
				
  • return: A unique ID for our record. It is in the format 999-ABCD and is fixed at 8  characters. The dash helps us identify the record faster, visually, if viewing the records  in a table.

  • access: public
